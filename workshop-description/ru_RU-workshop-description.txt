# Мод "Захват миссионеров" для Civilization VII

## Возможности
Возьмите под контроль религиозную войну с возможностью захватывать вражеских миссионеров во время войны!

### Основные функции:
- **Захват вражеских миссионеров**: Во время войны вы теперь можете захватывать вражеских миссионеров с помощью военных юнитов
- **Бой миссионеров**: Миссионеры имеют здоровье, которое должно быть сведено к нулю для их захвата
- **Открытые границы**: Настройте, нужны ли миссионерам открытые границы для входа на чужую территорию (выбирается во время настройки игры)

### Сбалансированная игровая механика
- Миссионеры больше не могут находиться в одной клетке с другими юнитами, что требует более стратегического позиционирования
- Каждый миссионер занимает свою собственную клетку

### Улучшенный религиозный геймплей
Этот мод добавляет стратегическую глубину религиозному соперничеству, позволяя цивилизациям напрямую противодействовать миссионерской деятельности с помощью военных действий во время войны.

## Как захватить миссионеров
1. Вы должны находиться в состоянии войны с цивилизацией, которой принадлежит миссионер
2. Атакуйте миссионера военным юнитом, чтобы снизить его здоровье до нуля
3. После поражения миссионер будет захвачен

## Совместимость
