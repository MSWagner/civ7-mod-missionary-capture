# 文明7傳教士捕獲模組

## 特性
在戰爭期間捕獲敵方傳教士，掌控宗教戰爭！

### 核心功能：
- **捕獲敵方傳教士**：戰爭期間，現在可以使用軍事單位捕獲敵方傳教士
- **傳教士戰鬥**：傳教士擁有生命值，必須將其降至零才能捕獲
- **開放邊界**：配置傳教士是否需要開放邊界才能進入外國領土（在遊戲設置時可選）

### 平衡遊戲機制
- 傳教士不能再與其他單位疊加，需要更加戰略性的定位
- 每個傳教士佔據自己的地塊

### 完整本地化
支持多種語言：
- 英語
- 德語 (Deutsch)
- 西班牙語 (Español)
- 法語 (Français)
- 義大利語 (Italiano)
- 日語 (日本語)
- 韓語 (한국어)
- 波蘭語 (Polski)
- 葡萄牙語 (Português)
- 俄語 (Русский)
- 中文（簡體和繁體）

### 增強宗教玩法
本模組通過允許文明在戰爭期間通過軍事行動直接對抗傳教活動，為宗教競爭增加了戰略深度。

## 如何捕獲傳教士
1. 你必須與擁有傳教士的文明處於戰爭狀態
2. 用軍事單位攻擊傳教士，將其生命值降至零
3. 一旦擊敗，傳教士就會被捕獲

## 兼容性
- 應與大多數其他模組兼容
